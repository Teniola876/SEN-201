## Project Overview
This repository contains the submission for the **SEN 201** assignment. The objective is to demonstrate the setup of a GitHub repository and the execution of a simple Python script using the Software Development Life Cycle (SDLC).

---

## SDLC Approach Followed
To complete this assignment, I followed the standard SDLC phases:

### 1. Planning & Requirements
* **Goal:** Create a program that displays a specific greeting.
* **Requirement:** The output must be exactly "Hello, World".
* **Tools:** Python 3, Git, and GitHub.

### 2. Design
* **Architecture:** A simple script-based architecture.
* **Logic:** Use the standard output function in Python (`print`) to display the required string.

### 3. Implementation (Coding)
* **Action:** Wrote the source code in `hello_world.py`.
* **Code:** `print("Hello, World")`

### 4. Testing
* **Unit Testing:** Ran the script locally using the command line to verify the output.
* **Result:** The terminal successfully displayed "Hello, World".

### 5. Deployment
* **Action:** Used Git to stage, commit, and push the code to this remote GitHub repository for submission.
